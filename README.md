# Sobre o Projeto

O **dB Konversor** é um aplicativo desenvolvido para simplificar cálculos envolvendo decibéis (dB), eliminando a necessidade de memorizar fórmulas complexas ou recorrer constantemente a calculadoras científicas.

O objetivo é disponibilizar uma ferramenta rápida, intuitiva e confiável que auxilie estudantes e profissionais durante aulas, laboratórios, provas e actividades do dia a dia.

O aplicativo foi desenvolvido pela **Kundiama Tecnologia e Inovação**, tendo como foco a simplicidade, precisão e facilidade de utilização.

---

# Principais Funcionalidades

✔ Conversão entre unidades em dB

✔ Conversões lineares ↔ logarítmicas

✔ Interface simples e intuitiva

✔ Cálculos instantâneos

✔ Funciona offline

# Como Executar

## 1. Clonar o repositório

```bash
git clone https://github.com/PauloTec/app_db_konversor.git
```

## 2. Entrar na pasta do projeto

```bash
cd app_db_konversor/db_konversor
```

## 3. Instalar as dependências

```bash
flutter pub get
```

## 4. Executar

```bash
flutter run
```
